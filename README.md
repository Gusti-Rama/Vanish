# Ketentuan
- Setiap Kelompok terdiri dari 2 mahasiswa membuat suatu aplikasi yang menggunakan algoritma kriptografi
- Kriteria aplikasi kriptografi sebagai berikut
    - Aplikasi Kriptografi pengirim pesan text/database, file dan gambar
    - Login aplikasi menggunakan algoritma Kriptografi (bisa hash atau yang lain seperti Jcrypto, AES, dll)
    - Pesan pada aplikasi berupa text di enkripsi menggunakan super enkripsi (algoritma kriptografi modern dan klasik)
    - Pesan Gambar dienkripsi menggunakan konsep steganografi 
    - File dienkripsi menggunakan konsep kriptografi yang lain
- Setiap kelompok membuat suatu penerapan algoritma atau aplikasi yang berbeda-beda dan dbuat laporan pengerjaan
- Tugas akhir dikumpulkan melalui link SPADA dan dipresentasikan pada  sebelum UAS
